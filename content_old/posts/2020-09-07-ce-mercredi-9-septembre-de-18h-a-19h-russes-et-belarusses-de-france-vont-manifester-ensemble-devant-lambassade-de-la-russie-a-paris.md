<h1 class="entry-title">
			Ce mercredi 9 septembre, de 18h à 19h,  Russes et Belarusses de France vont manifester ensemble devant l&rsquo;ambassade de la Russie à Paris.	</h1>	</header>
	
	<div class="entry-content">
				<p dir="ltr"><span class="il">Dans</span> <span class="il">les</span> <span class="il">prochains</span> <span class="il">jours</span> <span class="il">Alexandre</span> <span class="il">Loukachenko</span> <span class="il">va</span> <span class="il">se</span> <span class="il">rendre</span> <span class="il">à</span> <span class="il">Moscou</span>, <span class="il">pour</span> <span class="il">demander</span> <span class="il">au</span> <span class="il">Kremlin</span> <span class="il">une</span> <span class="il">aide</span> <span class="il">financière</span> <span class="il">et</span> <span class="il">politique</span>, <span class="il">afin</span> de <span class="il">préserver</span> <span class="il">son</span> <span class="il">pouvoir</span> <span class="il">dictatorial</span> <span class="il">et</span> <span class="il">renforcer</span> <span class="il">l</span>&lsquo;<span class="il">oppression</span> <span class="il">du</span> <span class="il">peuple</span> <span class="il">Belarus</span>.</p>
<p>Ce mercredi 9 septembre, de 18h <span class="il">à</span> 19h,  Russes <span class="il">et</span> Belarusses de France qui soutiennent <span class="il">les</span> transformations démocratiques <span class="il">et</span> la défense des droits humains <span class="il">dans</span> leurs pays vont manifester ensemble devant <span class="il">l</span>&lsquo;ambassade de la Russie <span class="il">à</span> Paris. La Russie <span class="il">et</span> le Bélarus sont en proie <span class="il">à</span> des  régimes autoritaires qui musèlent <span class="il">et</span> poursuivent <span class="il">l</span>&lsquo;opposition <span class="il">et</span> <span class="il">les</span> médias indépendants; des élections libres <span class="il">et</span> <span class="il">une</span> justice indépendante sont inexistantes.</p>
<p><span class="il">Les</span> Belarusses <span class="il">et</span> Russes de France qui soutiennent <span class="il">les</span> transformations démocratiques <span class="il">et</span> <span class="il">les</span> droits humains <span class="il">dans</span> leurs pays vont manifester ensemble <span class="il">pour</span> :</p>
<ul>
<li dir="ltr">exprimer leur désaccord <span class="il">au</span> financement <span class="il">du</span> dictateur <span class="il">belarusse</span> avec <span class="il">l</span>&lsquo;argent des contribuables russes;</li>
<li dir="ltr">exprimer leur solidarité avec le <span class="il">peuple</span> <span class="il">belarusse</span> qui <span class="il">se</span> bat pacifiquement contre la dictature;</li>
<li dir="ltr">exiger <span class="il">une</span> enquête internationale <span class="il">et</span> indépendante sur <span class="il">l</span>&#8217;empoisonnement de <span class="il">l</span>&lsquo;opposant Alexeï Navalny;</li>
<li dir="ltr">dire haut <span class="il">et</span> fort que <span class="il">les</span> Russes <span class="il">et</span> Belarusses sont contre le mensonge <span class="il">et</span> la violence d&rsquo;Etat, sont contre la corruption <span class="il">et</span> <span class="il">l</span>&lsquo;<span class="il">oppression</span>!</li>
</ul>
<p>Nous sommes ensemble <span class="il">pour</span> le choix indépendant <span class="il">et</span> souverain <span class="il">du</span> <span class="il">peuple</span> <span class="il">du</span> Bélarus <span class="il">et</span> sa liberté !</p>
<p><span class="il">Pour</span> votre liberté <span class="il">et</span> <span class="il">pour</span> la nôtre!</p>
